# dotfiles
clone with `--recurse-submodules` flag

## Applications

- Music: mpd/ncmpcpp
- File Manager: thunar + ranger
- Terminal: uxterm
- Web browser: firefox
- Editor: vim
- WM: dwm
- BitTorrent client: qbittorrent
- Password Manager: pass 

`mpd ncmpcpp thunar ranger firefox vim firefox pavucontrol qbittorrent password-store`
