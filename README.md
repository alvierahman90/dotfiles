## Applications

- Music: cmus
- File Manager: thunar + ranger
- Terminal: st
- Web browser: firefox
- Editor: vim
- WM: i3-gaps
- Volume control: pavucontrol
- Bars: polybar
- BitTorrent client: qbittorrent

`cmus thunar ranger firefox vim firefox pavucontrol qbittorrent`
